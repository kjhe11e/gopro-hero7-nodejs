Playing with a gopro hero 7.

# Setup

1. `npm install`

2. Create a `.env` file in the base directory and add the following:

```
MAC="{GOPRO_MAC_ADDRESS}"   # e.g. "aa:bb:cc:dd:ee:ff"
```
3. `npm run go`
